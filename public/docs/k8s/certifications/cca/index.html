        Exam Objects #

        CCA考试包括这些一般领域及其在考试中的权重：

        ● 架构： 20%

        ● 网络政策：18%

        ● 服务网格：16%

        ● 网络可观测性：10%

        ● 安装与配置

        ● 集群网

        ● eBPF

        ● BGP与外部组网

        架构：20%

        ● 了解Cilium在Kubernetes环境中的作用

        ● Cilium结构

        ● IP地址管理(IPAM)与Cilium

        ● Cilium成分的作用

        ● Datapath模型

        网络政策：18%

        ● 解释Cilium网络政策和意图

        ● 了解Cilium基于身份的网络安全模型

        ● 策略实施模式

        ● 策略规则结构

        ● Kubernetes网络策略与Cilium网络策略

        服务网格：16%

        ● 了解如何使用入口或网关API进行入口路由

        ● 服务网格用例

        ● 了解网关API相对于入口API的好处

        ● 使用Cilium加密传输中的流量

        ● 基于边车与无边车架构

        网络可观测性：10%

        ● 了解Hubble的观测能力

        ● 启用第七层协议可见性

        ● 了解如何从命令行或Hubble用户界面使用Hubble
